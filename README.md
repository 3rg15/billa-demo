# Billa Demo Website

A responsive demo website built for **Billa Albania**, showcasing store locations, contact information, and a modern design suitable for portfolio presentation.

## 🚀 Features
- Fixed navbar with responsive mobile menu
- Store sections with Google Maps embeds
- Image sliders for showroom photos
- Contact page with styled icons for address, phone, email, and social media
- Clean, modern typography and alternating section backgrounds
